# Holberton School Machine Learning

This repository contains projects and exercises for learning **Machine Learning** concepts using Python.

## Structure

- `pipeline/pandas/` → Data manipulation tasks with pandas
- `math/plotting/` → Visualization tasks with matplotlib and numpy

## Requirements

- Python 3.9
- numpy 1.25.2
- matplotlib 3.8.3
- pycodestyle 2.11.1

All files:
- Start with `#!/usr/bin/env python3`
- Are executable
- Follow PEP8 style guidelines
- Include documentation for modules, classes, and functions
